# Wedding Invitation

## 설치

프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따르세요.

1. 먼저 이 프로젝트를 복제합니다.
```sh
git clone https://github.com/windragon0807/wedding-invitation.git
```

2. 프로젝트 폴더로 이동합니다.
```sh
cd wedding-invitation
```

3. 프로젝트 종속성을 설치합니다.
```sh
# npm
npm install

# yarn
yarn install
```

## 실행

프로젝트를 실행하려면 다음 명령을 사용하세요.
```sh
# npm
npm run start

# yarn
yarn start
```
